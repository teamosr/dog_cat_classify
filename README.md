# dog_cat_classify
This is the most classic introductory mini-project for GNN
# About the related data
数据集包含在https://www.kaggle.com/c/dogs-vs-cats/data 该链接中，访问该链接可以下载相关数据用于实验
# Code Description
由于时间原因和个人电脑配置原因，本人自行并未使用全部的数据集，而是使用了其中的部分数据集，如果您有条件可以尝试使用全部数据集来进行使用，如果仅仅是用来学习，个人建议使用部分数据集即可，重点是掌握GNN的整体流程
# Visual operation
在学者在进行的过程中会保存生成一个自己训练好的模型，类似dog_cat.h5的文件，为了该项目不再繁琐的使用代码来上传图片来进行测试，所以读者可以使用可视化来进行测试，具体使用streamlit来实现，首先读者需要在 Anaconda Prompt 里安装 Streamlit，然后粘贴我所提供的python代码，确保这个py文件（假设名字为app.py）和您训练好的模型在同一个文件夹中，然后进入Anaconda Prompt中运行streamlit run app.py即可进入可视化界面
