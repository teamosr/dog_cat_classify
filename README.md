# dog_cat_classify
This is the most classic introductory mini-project for GNN
# About the related data
数据集包含在https://www.kaggle.com/c/dogs-vs-cats/data 该链接中，访问该链接可以下载相关数据用于实验
# Code Description
由于时间原因和个人电脑配置原因，本人自行并未使用全部的数据集，而是使用了其中的部分数据集，如果您有条件可以尝试使用全部数据集来进行使用，如果仅仅是用来学习，个人建议使用部分数据集即可，重点是掌握GNN的整体流程
